# [Kamilek7.github.io/Niedziele-Handlowe](https://kamilek7.github.io/Niedziele-Handlowe/)

Od roku 2018 Polaków męczą następujące pytania:
* Czy dzisiaj jest niedziela handlowa?
* Czy następna niedziela jest handlowa?
* Handel?
* Niedziela handlująca handlem?

Na te wszystkie pytania można odpowiedzieć prostym **TAK**/**NIE**, bez potrzeby czytania długich artykułów o genezie zakazu handlu w Polsce.<br>
I właśnie po to powstał ten projekt, od poznania odpowiedzi na te nurtujące pytania dzieli cię tylko [jedno klinięcie.](https://kamilek7.github.io/Niedziele-Handlowe)

*dane pobierane są ze strony [kalendarzswiat.pl](https://www.kalendarzswiat.pl/)*
